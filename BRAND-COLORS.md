# A-One Agency — Brand Color System

## Tactical Control Theme

---

## 1. NEUTRAL (Серые/Фоны)

Основа интерфейса. Чистые, матовые поверхности.

| Название | HEX | RGB | Использование |
|----------|-----|-----|---------------|
| **Background** | `#0B0C10` | rgb(11, 12, 16) | Основной фон страницы |
| **Surface** | `#121418` | rgb(18, 20, 24) | Вторичный фон, секции |
| **Card** | `#1F2833` | rgb(31, 40, 51) | Карточки, поповеры |
| **Border** | `#1F2833` | rgb(31, 40, 51) | Границы, разделители |
| **Border Light** | `#2D3748` | rgb(45, 55, 72) | Hover-состояния границ |

### Текст (на темном фоне)
| Название | HEX | RGB | Использование |
|----------|-----|-----|---------------|
| **Heading** | `#FFFFFF` | rgb(255, 255, 255) | Заголовки H1-H6 |
| **Body** | `#9CA3AF` | rgb(156, 163, 175) | Основной текст, описания |
| **Muted** | `#6B7280` | rgb(107, 114, 128) | Вспомогательный текст |

---

## 2. PRIMARY — Зеленый (Кнопки, Блоки)

Основной акцентный цвет. Кнопки, активные состояния, важные блоки.

| Название | HEX | RGB | Использование |
|----------|-----|-----|---------------|
| **Primary** | `#10B981` | rgb(16, 185, 129) | Основные кнопки, CTA |
| **Primary Hover** | `#059669` | rgb(5, 150, 105) | Hover-состояние |
| **Primary Light** | `#34D399` | rgb(52, 211, 153) | Легкий акцент |
| **Primary BG** | `rgba(16, 185, 129, 0.1)` | — | Фон иконок, бейджей |
| **Primary Border** | `rgba(16, 185, 129, 0.3)` | — | Границы акцентных элементов |

### Glow Effect
```css
--shadow-glow-green: 0 0 10px rgba(16, 185, 129, 0.3);
--shadow-glow-green-strong: 0 0 20px rgba(16, 185, 129, 0.4);
```

---

## 3. DATA — Голубой Неон (Цифры, Графики)

Для визуализации данных, метрик, статистики.

| Название | HEX | RGB | Использование |
|----------|-----|-----|---------------|
| **Neon Cyan** | `#66FCF1` | rgb(102, 252, 241) | Цифры, проценты, KPI |
| **Neon Cyan Light** | `#9EFFF8` | rgb(158, 255, 248) | Подсветка данных |
| **Neon Cyan Dark** | `#45A29E` | rgb(69, 162, 158) | Вторичные данные |

### Glow Effect
```css
--shadow-glow-cyan: 0 0 10px rgba(102, 252, 241, 0.3);
--shadow-glow-cyan-strong: 0 0 20px rgba(102, 252, 241, 0.5);
```

### Применение:
- Большие цифры в карточках (49 000 ₸, +340%, 15 мин)
- Графики и диаграммы
- Прогресс-бары
- Счетчики и таймеры
- Статусы "онлайн"

---

## 4. SYSTEM — Уведомления и Статусы

| Тип | Название | HEX | RGB | Использование |
|-----|----------|-----|-----|---------------|
| **Success** | Teal | `#14B8A6` | rgb(20, 184, 166) | Успешные действия |
| **Warning** | Amber | `#F59E0B` | rgb(245, 158, 11) | Предупреждения |
| **Error** | Red | `#EF4444` | rgb(239, 68, 68) | Ошибки |
| **Info** | Blue | `#3B82F6` | rgb(59, 130, 246) | Информация |

### Фоны уведомлений
```css
--success-bg: rgba(20, 184, 166, 0.1);
--warning-bg: rgba(245, 158, 11, 0.1);
--error-bg: rgba(239, 68, 68, 0.1);
--info-bg: rgba(59, 130, 246, 0.1);
```

## Полная палитра (CSS Variables)

```css
:root {
  /* ========== NEUTRAL ========== */
  --background: #0B0C10;
  --surface: #121418;
  --card: #1F2833;
  --border: #1F2833;
  --border-light: #2D3748;

  /* Text */
  --text-heading: #FFFFFF;
  --text-body: #9CA3AF;
  --text-muted: #6B7280;

  /* ========== PRIMARY (Green) ========== */
  --primary: #10B981;
  --primary-hover: #059669;
  --primary-light: #34D399;
  --primary-foreground: #0B0C10;

  /* ========== DATA (Neon Cyan) ========== */
  --data: #66FCF1;
  --data-light: #9EFFF8;
  --data-dark: #45A29E;

  /* ========== SYSTEM ========== */
  --success: #14B8A6;
  --warning: #F59E0B;
  --error: #EF4444;
  --info: #3B82F6;

  /* ========== SHADOWS ========== */
  --shadow-glow-green: 0 0 10px rgba(16, 185, 129, 0.3);
  --shadow-glow-cyan: 0 0 10px rgba(102, 252, 241, 0.3);
}
```

## Примеры использования

### Кнопки (Primary Green)
```jsx
<button className="bg-primary text-primary-foreground hover:bg-primary-hover">
  Начать сейчас
</button>
```

### Данные (Neon Cyan)
```jsx
<span className="text-data text-4xl font-bold">49 000 ₸</span>
<span className="text-data-dark">в месяц</span>
```

### Карточки
```jsx
<div className="bg-card border border-border hover:border-primary">
  ...
</div>
```

### Уведомления
```jsx
<div className="bg-success/10 text-success border border-success/30">
  Успешно сохранено
</div>

<div className="bg-error/10 text-error border border-error/30">
  Произошла ошибка
</div>
```
